# MoradBot — System Diagrams

هذا المجلد يحتوي على مخططات النظام بصيغة Mermaid.

---

## كيفية العرض

### GitHub / GitLab
الـ Mermaid يُعرض تلقائياً داخل ملفات `.md` على GitHub وGitLab.
افتح أي ملف مباشرةً في واجهة الويب.

### VS Code
ثبّت إضافة [Markdown Preview Mermaid Support](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-mermaid)،
ثم اضغط `Cmd+Shift+V` لعرض الـ Preview.

### mermaid.live
انسخ أي كتلة Mermaid وألصقها مباشرةً في [mermaid.live](https://mermaid.live).
