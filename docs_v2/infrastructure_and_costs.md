### 2.5 OpenRouter (تكلفة الذكاء الاصطناعي)

لا رسوم ثابتة — pay-as-you-go بالكامل.

#### النماذج المستخدمة في MoradBot

| النموذج | الدور | Input ($/1M token) | Output ($/1M token) |
|---------|-------|--------------------|--------------------|
| `google/gemini-2.0-flash` | **Primary — الإنتاج (الباقات المدفوعة)** | ~$0.10 | ~$0.40 |
| `gemini-2.0-flash-exp:free` | الباقة المجانية فقط | **$0** | **$0** (rate limited) |
| `openai/gpt-4o-mini` | Fallback 1 (عند فشل Gemini) | $0.15 | $0.60 |

> **المصدر:** [openrouter.ai/pricing](https://openrouter.ai/pricing) — تحقق دورياً من الأسعار.

#### حساب تكلفة الرد الواحد

```
كل رد روبوت يستهلك تقريباً:
  Input:  1,400 token  (system prompt ~400 + FAQ context ~600 + محادثة ~400)
  Output:   200 token  (رد عربي قصير)

تكلفة رد واحد — Gemini 2.0 Flash (الإنتاج):
  Input:  1,400 × ($0.10 / 1,000,000) = $0.000140
  Output:    200 × ($0.40 / 1,000,000) = $0.000080
  ─────────────────────────────────────────────────
  المجموع/رد:              ≈ $0.000220
  أو: ~0.82 ريال لكل 1,000 رد

تكلفة رد واحد — GPT-4o Mini (Fallback — نادر):
  Input:  1,400 × ($0.15 / 1,000,000) = $0.000210
  Output:    200 × ($0.60 / 1,000,000) = $0.000120
  ─────────────────────────────────────────────────
  المجموع/رد:              ≈ $0.000330
  أو: ~1.24 ريال لكل 1,000 رد

التكلفة المُدمجة (95% Gemini + 5% GPT-4o Mini):
  = 0.95 × $0.000220 + 0.05 × $0.000330 = $0.000226/رد
  أو: ~0.85 ريال لكل 1,000 رد
```

> **ملاحظة:** ≥95% من الردود عبر Gemini 2.0 Flash. GPT-4o Mini يُستخدم فقط عند فشل Gemini.

## القسم 5 — استراتيجيات تخفيض التكاليف

### تخفيض تكلفة OpenRouter

| الاستراتيجية | التوفير المتوقع |
|-------------|----------------|
| تفعيل Gemini context caching (90% خصم على tokens مكررة) | 30-50% إذا كان system prompt ثابت |
| تقليل حجم prompt بـ FAQ context compression | 15-20% |
| الانتقال لـ `gemini-2.0-flash-lite` عند ارتفاع التكاليف | 25% أقل (مع مراقبة جودة الردود) |

### تخفيض تكلفة Supabase

| الاستراتيجية | الفائدة |
|-------------|--------|
| أرشفة المحادثات القديمة (> 90 يوم) | حفاظ على database < 8GB |
| استخدام Connection Pooler (PgBouncer) | تقليل Compute usage |
| Soft delete بدلاً من Hard delete | لا فائدة في التكلفة، لكن يحفظ البيانات |

### Cloudflare Workers — لا تكلفة إضافية حتى

Workers يغطي 10M طلب/شهر:
  10,000,000 ÷ (3 طلبات/رد × 3,700 رد/متجر) ≈ 900 متجر
لن تحتاج ترقية قبل الوصول لـ ~900 متجر. عند 1,000 متجر: تجاوز بسيط (~10%) بتكلفة $0.30/مليون إضافي.
