# CNN Layers

> **View online:** https://deepbox.dev/examples/27-cnn-layers

Demonstrates convolutional neural network layers: Conv1d, Conv2d, MaxPool2d, and AvgPool2d.

## Deepbox Modules Used

| Module            | Features Used                                    |
| ----------------- | ------------------------------------------------ |
| `deepbox/ndarray` | tensor, reshape, GradTensor                      |
| `deepbox/nn`      | Conv1d, Conv2d, MaxPool2d, AvgPool2d, Sequential |

## Usage

```bash
npm run example:27
```

## Output

- Console output demonstrating convolution and pooling operations with shape transformations
